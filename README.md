# ResolveX - Smart Complaint Management System

A full-stack web application designed to streamline complaint registration, tracking, and resolution within organizations and institutions. ResolveX enables users to submit complaints, monitor their status, and receive updates, while administrators can efficiently manage and resolve complaints through a dedicated dashboard.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Shrutika-khade/Smart-Complaint-Management-System.git
```

### Backend Setup

```bash
cd Complaint-Management-System
mvn spring-boot:run
```

### Configure Database

Update application.properties with your MySQL credentials.

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/complaint_db
spring.datasource.username=root
spring.datasource.password=your_password
```

### Run Application

```bash
mvn spring-boot:run
```
