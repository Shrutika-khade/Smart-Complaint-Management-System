# Smart Complaint Management System

A full-stack web application developed to streamline complaint registration, tracking, and resolution within an organization or institution. The system enables users to submit complaints, track their status, and receive updates, while administrators can efficiently manage and resolve complaints through a dedicated dashboard.

## Features

### User Features
- User Registration & Login
- JWT Based Authentication
- Submit New Complaints
- Track Complaint Status
- View Complaint History
- Profile Management

### Admin Features
- Admin Dashboard
- Manage Complaints
- Update Complaint Status
- Manage Users
- Complaint Analytics
- Department-wise Complaint Management

### Smart Features
- Auto Priority Detection
- Role-Based Access Control
- Secure Password Encryption using BCrypt
- Real-Time Complaint Tracking

## Tech Stack
